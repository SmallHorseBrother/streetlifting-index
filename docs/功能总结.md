# 街健力量指数网站功能总结

## 项目概述

街健力量指数（Streetlifting Index）是一个社区驱动的开源项目，旨在为街头健身爱好者提供科学、公平、透明的力量评估标准。项目由北京大学博士生枭马葛创建，通过收集和分析大量真实的训练数据，建立体重标准化的力量评估体系。

## 核心功能模块

### 1. 首页（Home Page）

**主要功能：**
- 项目介绍和核心理念展示
- 快速入口导航：
  - 计算我的力量分
  - 贡献我的数据
  - 寻找单杠
- 实时统计数据展示：
  - 总数据收录数量
  - 公式最后更新时间
- 社区动态展示（最近提交的数据）
- 社交媒体链接（B站、抖音）

**特色：**
- 响应式设计，支持移动端和桌面端
- 渐变背景和现代化UI设计
- 清晰的信息层次和视觉引导

### 2. 力量计算器（Calculator）

**支持的运动类型：**
- 💪 **负重引体向上**（使用枭马葛公式，满分500分）
- 💪 **负重双杠臂屈伸**（使用枭马葛公式，满分500分）
- 🏋️ **实力推**（基于卧推DOTS公式换算，满分500分）
- 🏋️ **杠铃弯举**（基于卧推DOTS公式换算，满分500分）
- 🏋️ **深蹲**（使用国际标准DOTS公式）
- 🏋️ **卧推**（使用国际标准DOTS公式）
- 🏋️ **硬拉**（使用国际标准DOTS公式）

**计算模式：**

1. **正向计算**
   - 输入：做组重量/附加负重、完成次数、性别、体重、动作质量
   - 输出：估算1RM、力量分、力量等级、RM对照表

2. **反推做组重量**
   - 输入：目标1RM、完成次数、性别、体重、动作质量
   - 输出：完成指定次数所需的做组重量、力量分

3. **反推次数**
   - 输入：目标1RM、做组重量、性别、体重、动作质量
   - 输出：可完成的次数、力量分

4. **做组极限估算**（仅限上肢类运动）
   - 输入：做组重量、组数、每组次数、休息时长、第一组/最后一组RIR或实际次数
   - 输出：当天极限负重1RM、力量分

**动作质量惩罚机制：**
- **比赛级标准**：无惩罚（0kg）
- **轻微借力**：2-5kg惩罚（用户自评）
- **严重借力**：5-20kg惩罚（用户自评）
- **超严重借力**：20-50kg惩罚（用户自评）

**力量等级评定：**
- 世界级/世界一线（🏆）
- 国内顶级/国内一线（🥇）
- 大佬（💪）
- 爱好者水平（🔥）
- 菜就多练（🌱）

**其他功能：**
- RM对照表（显示1-16次对应的做组重量和百分比）
- 实时公式系数获取（从数据库动态加载最新公式）
- 详细的计算结果展示

### 3. 数据提交（Submit）

**提交内容：**
- 运动类型（负重引体向上/负重双杠臂屈伸）
- 基础信息：性别、体重
- 训练数据：附加负重、完成次数
- 动作质量评估（含惩罚重量设置）
- 引体类型（正手/反手，仅限负重引体）
- 用户信息（选填）：姓名/社媒账号、视频链接

**数据验证：**
- 必填字段验证
- 数值范围验证（次数限制1-10次）
- 惩罚重量范围验证

**提交流程：**
- 表单填写 → 数据验证 → 提交到数据库 → 跳转成功页面

### 4. 社区数据展示（Data）

**数据展示方式：**
- **卡片视图**：美观的卡片布局，展示关键信息
- **表格视图**：详细的数据表格，适合快速浏览和对比

**筛选功能：**
- 运动类型（负重引体/负重臂屈伸）
- 性别（男性/女性）
- 引体类型（正手/反手）
- 动作质量（比赛级/轻微借力/严重借力/超严重借力）
- 是否有视频
- 用户名搜索

**排序功能：**
- 按时间排序（最新优先）
- 按绝对重量排序（1RM从高到低）
- 按力量分排序（力量分从高到低）

**数据展示内容：**
- 用户信息（姓名/匿名）
- 运动类型和引体类型
- 性别、体重、负重、次数
- 估算1RM
- 力量分（基于最新公式计算）
- 动作质量标签
- 训练视频链接（如有）
- 提交时间

**统计信息：**
- 筛选结果总数
- 包含视频的数据数量
- 负重引体数据数量
- 负重臂屈伸数据数量

## 技术特性

### 前端技术栈
- **框架**：Next.js（React）
- **UI组件库**：shadcn/ui
- **样式**：Tailwind CSS
- **状态管理**：React Hooks
- **图标**：Lucide React

### 后端服务
- **数据库**：Supabase（PostgreSQL）
- **存储**：Supabase Storage（图片存储）
- **认证**：Supabase Auth（匿名访问）

### 核心算法

1. **1RM估算算法**
   - Epley公式：`Total_1RM = Total_Weight × (1 + 0.0333 × Reps)`
   - Brzycki公式：`Total_1RM = Total_Weight × (36 / (37 - Reps))`
   - Lombardi公式：`Total_1RM = Total_Weight × (Reps ^ 0.1)`
   - 最终估算：三公式平均值

2. **力量系数公式（枭马葛公式）**
   - 4阶多项式：`C(W) = a·W⁴ + b·W³ + c·W² + d·W + e`
   - 男女分别计算
   - 动态从数据库加载最新系数

3. **DOTS公式**（用于力量三项）
   - 国际标准DOTS系数计算
   - 男女分别计算

4. **做组极限估算算法**
   - 基于组数、次数、休息时长
   - 考虑第一组和最后一组的表现
   - 使用强度衰减模型
